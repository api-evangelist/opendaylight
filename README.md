# OpenDaylightThis is a repo for managing machine readable artifacts for the [OpenDaylight APIs](http://www.opendaylight.org/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [OpenDaylight APIs](http://www.opendaylight.org/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.opendaylight.org/)- [Developer](http://www.opendaylight.org/)- [Twitter](https://twitter.com/OpenDaylightSDN)- [GitHub](https://github.com/opendaylight)The [APIs.json](https://github.com/api-evangelist/opendaylight/blob/master/apis.json) index provides links to all the relevant links for the OpenDaylight APIs, including the OpenAPI and Postman if they are present.
